# ProjectX

Requirements
---

Node.js version 10

Installation
---

1. Install Yeoman and Chisel globally. In your command line type:
``npm install -g yo generator-chisel``

2. Clone repository
3. Run ``yarn``
4. Run ``npm run build`` and ``npm run dev``
